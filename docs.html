JsBarcode

Documentation

Format

Select which barcode type to use.
Please check the wikipage of the different
barcode types for more information.

JsBarcode("#barcode", "123456789012", {
  format: "EAN13"
});
JsBarcode("#barcode", "123456789012", {
  format: "CODE39"
});

Height

The height of the barcode. 

JsBarcode("#barcode", "Taller barcode", {
  height: 150
});
JsBarcode("#barcode", "Small barcode", {
  height: 25
});

Width

The width option is the width of a single bar. 

JsBarcode("#barcode", "Smallest width", {
  width: 1
});
JsBarcode("#barcode", "Wider barcode", {
  width: 3
});

Text Position

Set the vertical position of the text.
Can be bottom / top.

JsBarcode("#barcode", "Top text", {
  textPosition: "top"
});

Text Margin

Set the space between the barcode and the text.

JsBarcode("#barcode", "Text margin", {
  textMargin: 25
});

Text Align

Set the horizontal alignment of the text.
Can be left / center / right.

JsBarcode("#barcode", "Left text", {
  textAlign: "left"
});

Background

Set the background of the barcode.

JsBarcode("#barcode", "Blue background", {
  background: "#ccffff"
});

Line Color

Set the color of the bars and the text.

JsBarcode("#barcode", "Red lines", {
  lineColor: "#990000"
});

Font

Define the font used for the text in the generated barcode.
This can be any default font or a font defined by a @font-face rule.

JsBarcode("#barcode", "Fantasy font", {
  font: "fantasy"
});

Font Size

Set the size of the text in pixels

JsBarcode("#barcode", "Bigger text", {
  fontSize: 40
});

Font Options

With fontOptions you can add bold or italic text to the barcode.

JsBarcode("#barcode", "Bold text", {
  fontOptions: "bold"
});
JsBarcode("#barcode", "Italic text", {
  fontOptions: "italic"
});
JsBarcode("#barcode", "Both options", {
  fontOptions: "bold italic"
});

Margins

Set the space margin around the barcode.
If nothing else is set,
all side will inherit the margins property but can
be replaced if you want to set them separably.

JsBarcode("#barcode", "Bigger margins", {
  margin: 30,
  background: "#dddddd"
});
JsBarcode("#barcode", "Left/Top margin", {
  marginLeft: 30,
  marginTop: 50,
  background: "#dddddd"
});
